Course Scheduler App

The Goal: To streamline the complex process of class planning for University of Toledo students and academic advisors by providing an automated, intelligent course scheduling tool that eliminates manual data entry and resolves conflicts.

What You Did: As a full-stack developer, I engineered Python scripts using PDFPlumber to automatically parse dense UToledo degree flowcharts. I then developed a web application using React and Flask and implemented intelligent algorithms to automatically detect and prevent time conflicts, while also allowing students to apply over 5 custom preferences (e.g., "no Friday classes").

What Happened Because of It: This application reduced manual scheduling time by over 70% during mock tests, directly improving efficiency for both students and advising staff by automating data extraction and conflict resolution.

AI-Powered Voice Assistant on Raspberry Pi

The Goal: To build a low-power, always-on conversational AI assistant that could operate independently on an embedded system (Raspberry Pi), providing quick, hands-free interaction.

What You Did: I configured Raspberry Pi GPIO pins for visual feedback and integrated USB audio peripherals. I then developed Python scripts to handle real-time speech-to-text and text-to-speech processing, all while interfacing with the Gemini API for advanced conversational AI.

What Happened Because of It: The assistant achieved a sub-1-second response latency, offering a fluid and highly responsive user experience. This project successfully demonstrated the practical application of cloud AI on a resource-constrained edge device, showcasing capabilities in embedded systems and hardware control.

ReviewGenie (AI-Powered Review Tool)

The Goal: To simplify the user review process for businesses and provide them with AI-driven insights from customer feedback, making data more actionable and accessible.

What You Did: I developed an embeddable widget designed for easy integration into business websites. This widget utilized AI to process user reviews, and I designed a responsive React-based frontend to ensure a seamless and intuitive user experience across various devices.

What Happened Because of It: The tool streamlined review collection and successfully provided AI-driven business insights, empowering companies to better understand customer sentiment and improve decision-making based on user feedback.
